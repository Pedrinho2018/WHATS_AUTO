# Diagramas vetoriais

Arquivos SVG prontos para importar em Figma, Illustrator, Inkscape, diagrams.net/draw.io, LibreOffice Draw ou outro editor vetorial.

## Arquivos

| Arquivo | Conteudo |
|---|---|
| `arquitetura-geral.svg` | Componentes principais: frontend, backend, banco, Evolution, n8n e Typebot |
| `modelo-dados-er.svg` | Entidades principais e relacionamentos |
| `fluxo-mensagem-inbound.svg` | Sequencia de mensagem recebida do WhatsApp |
| `fluxo-resposta-operador.svg` | Sequencia de resposta enviada pelo operador |
| `estado-ticket.svg` | Estados de uma conversa/ticket |

## Como usar

### Figma

Use `File -> Place image` ou arraste o SVG para a tela. Depois, se quiser editar as formas, selecione o SVG importado e use `Flatten`/`Outline stroke` conforme a necessidade do arquivo.

### diagrams.net / draw.io

Use `File -> Import From -> Device` e selecione o SVG. Outra opcao e arrastar o arquivo para o canvas.

### Inkscape ou Illustrator

Abra o SVG diretamente. Textos, linhas e caixas foram criados como elementos vetoriais.

## Observacao

Os diagramas em `docs/UML.md` continuam em Mermaid para leitura rapida no GitHub. Esta pasta existe para edicao visual em ferramentas externas.
